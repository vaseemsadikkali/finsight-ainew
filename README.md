# FinSight AI 📈🤖

FinSight AI is an AI-powered market analysis and news platform that helps users track stocks, cryptocurrencies, and market trends in real time.

## 🌐 Live Demo

**Website:** https://finsight-ainew.onrender.com/

Try the application online without installing anything.

## Features

* 📊 Real-time market data
* 📰 Latest market news
* ₿ Cryptocurrency tracking
* 📈 Stock price monitoring
* 🤖 AI-powered market insights
* 🔍 Symbol-based search
* 📱 Responsive design
* ⚡ Fast and lightweight interface

## Technologies Used

* Node.js
* Express.js
* JavaScript
* HTML5
* CSS3
* Yahoo Finance API
* REST APIs

## Installation

### Clone the repository

```bash
git clone https://github.com/vaseemsadikkali/finsight-ainew.git
cd finsight-ainew
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
